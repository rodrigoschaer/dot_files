## Useful maps
- `%` - creates a new file
- `d` - creates a new dir
- `shift + i` - help menu toggle
- `p` - preview the file
- `shift + d` - attempt to delete a file/directory  

## Moving file(f) to a directory(t)
1 - `mt` - target (or select) the directory as a file
2 - `mf` - mark a file
3 - `mc` - copy market file to target marked directory
4 - `D` - delete old file
